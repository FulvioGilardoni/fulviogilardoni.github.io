Requisiti

  • Windows 10/11 (x64)
  • Nessuna connessione internet richiesta

Download

Scarica dalla pagina Releases il file camt2csv-demo-win-x64.zip. Verifica l’hash SHA256 riportato nella Release.

Installazione

  1. Estrai lo ZIP in una cartella (es. C:\CAMT2CSV\).
  2. Apri il Prompt dei comandi in quella cartella.

Uso

camt2csv.exe input.xml output.csv
camt2csv.exe C:\camt\ C:\csv\

Opzioni utili

  • --out <file|cartella>: percorso di destinazione
  • --sep ;|,: separatore CSV (default ;)
  • --culture it-IT|en-US: formattazione numeri/date
  • --single: forza modalità singolo file

Limitazioni DEMO

Elabora fino a 20 righe per file e lo indica a fine esecuzione.

SmartScreen (file non firmato)

Se Windows mostra un avviso: clicca Ulteriori informazioni → Esegui comunque. Assicurati di aver scaricato dalla pagina ufficiale e verifica la firma SHA256 pubblicata nella Release.

Formati supportati

Input: CAMT.053 / CAMT.054 (ISO 20022). Output: CSV con separatore ;.
