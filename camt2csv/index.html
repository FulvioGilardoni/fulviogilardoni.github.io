Come si usa (30 secondi)

1) Scarica e scompatta lo ZIP
2) Apri "Prompt dei comandi" nella cartella
3) Esegui:
   camt2csv.exe input.xml output.csv
   - oppure -
   camt2csv.exe C:\camt\ C:\csv\

Caratteristiche principali

  • Parsing CAMT.053 / CAMT.054 (ISO 20022) → CSV
  • Funzionamento offline (nessuna rete, nessuna telemetria)
  • Compatibile con Excel/LibreOffice e gestionali (Teamsystem, Zucchetti, Buffetti, Profis, Banana, ERP)
  • Modalità singolo file o batch su cartella

FAQ

Serve Internet? No, il software opera totalmente offline.

Quante righe elabora la DEMO? Fino a 20 righe per file (per test). Versione completa senza limiti.

Il file EXE è firmato? Al momento no. Se Windows mostra SmartScreen: clicca Ulteriori informazioni → Esegui comunque. Verifica l’integrità con l’hash SHA256 indicato nella Release.

Quali formati supporta? CAMT.053 e CAMT.054 (ISO 20022); output CSV.
