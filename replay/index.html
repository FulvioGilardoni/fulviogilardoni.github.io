Replay – Understand what changed in your data

Essential documentation (English)

Replay compares successive versions of CSV and Excel (.xlsx) files, highlighting only what changed: added, removed and modified rows. No more manual comparisons between spreadsheets. Upload a baseline, then upload new versions and instantly see the differences.

Why Replay

Structured data changes over time: price lists, inventories, contact databases, exports from management systems. Replay shows you exactly what changed between one version and the next, without having to compare row by row.

How it works

  1. Create a project and upload a CSV or Excel file as a baseline.
  2. When data changes, upload the new version.
  3. Replay analyses the differences and displays them clearly: added, removed and modified rows.
