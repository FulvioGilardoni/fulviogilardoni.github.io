1. Descrizione del Servizio

Replay è un servizio web che consente di confrontare versioni successive di file di dati strutturati (CSV, XLSX) per identificare le differenze tra una versione e la successiva. Il Servizio:

  • Permette il caricamento di file tabellari
  • Calcola automaticamente le differenze tra versioni
  • Presenta i risultati in forma visuale e scaricabile (PDF)
  • Non archivia permanentemente i file caricati
