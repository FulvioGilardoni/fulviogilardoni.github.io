TableDealer respects user privacy. This application does not collect, store, or share personal data.

TableDealer does NOT collect any personal data, including but not limited to:

  • Name and surname
  • Email address
  • Phone number
  • Location data
  • Contacts
  • Persistent user identifiers
  • Payment information
  • Usage data for profiling or marketing purposes

The app uses cloud services (Firebase) solely for real-time technical synchronization between players during a game session.

  • Table state
  • Card distribution
  • Hand status (Preflop, Flop, Turn, River)

Such data:

  • Does not personally identify users
  • Is not used for tracking
  • Is not used for analytics or advertising
  • Is not stored beyond the game session duration

  • Payments are handled directly by Google Play

TableDealer does not share personal data with third parties. Third-party services (such as Firebase) are used strictly as technical infrastructure.

6. Security

Since no personal data is collected, no sensitive user data is stored. All communications use secure channels provided by the underlying services.
