How to write an NFC tag with a counter

This guide shows how to write an NFC tag so that it contains a counter managed by the TagTally app.

Requirements

  • Phone with NFC enabled.
  • NFC tag (for example NTAG 213, 215 or 216) formatted as NDEF.
  • TagTally app installed.

Steps

  1. Open TagTally on your phone.
  2. Tap the phone on the NFC tag.
  3. If the tag is empty, the app will ask to initialize the counter.
  4. Choose the initial value and, optionally, a tag name.
  5. Confirm to write the counter to the tag.

Updating the counter

Later, when you tap the tag again with TagTally:

  • Read the current value.
  • Increase or decrease the counter.
  • Set a specific value if needed.
