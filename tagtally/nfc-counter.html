NFC counter: store a value inside the tag

An NFC counter is a number saved directly inside an NFC tag using the NDEF format. The value lives in the tag, not in the phone or in the cloud.

Why use an NFC counter

  • The value follows the physical object.
  • Multiple phones can read and update the same counter.
  • No internet connection or account is required.

Typical use cases

  • Small warehouses and labs.
  • Schools, clubs and associations.
  • Shared kits, tools and consumables tracking.

Using an NFC counter with TagTally

  1. Tap your phone on the NFC tag.
  2. TagTally reads the current value stored in the tag.
  3. You can increase, decrease or set an exact new value.
  4. The updated value is written back to the tag and can be read by other phones using TagTally.
