NTAG 213 NFC counter example

The NTAG 213 is a common NFC tag used in cards, stickers and labels. With TagTally you can use it as an NFC counter, storing the value directly in NDEF.

Why NTAG 213 for counters

  • Widely available and inexpensive.
  • Enough memory to store a counter and basic metadata.
  • Compatible with most NFC-enabled phones.

Using NTAG 213 with TagTally

  1. Place the NTAG 213 tag near your phone.
  2. Open the TagTally app.
  3. Initialize the tag with a starting counter.
  4. Each tap lets you read and update the counter value stored in the NDEF record.
