Privacy Policy TagTally

Last update: November 2025

1. General Information

TagTally is an application that uses NFC technology to read and update a counter stored directly on compatible NFC tags (NDEF). The app is designed to work offline, without accounts and without cloud services.

2. Types of Data Processed

No personal data is collected, stored, or transmitted.

The app handles only:

  • Data stored on the NFC tag: for example, an integer (counter) and, optionally, a tag name chosen by the user.
  • Local data on the device: basic local usage history and app preferences (e.g., theme or read-only mode).

This data remains only on the device and/or on the NFC tag and is not sent to any external server or third party.

3. App Permissions

PermissionPurposeData transmission
NFCReading and writing the counter on the tagNo transmission
Vibration (Haptic feedback)Tactile feedback during useNo transmission
(Optional) Local file accessManual saving/export of CSV filesNo transmission

TagTally does not access the camera, microphone, contacts, or location.

4. Internet Connection

TagTally does not use the Internet for any operational feature of the app.

The connection is used only to manage in-app purchases and license verification through Google Play.
No data related to NFC tags, app usage, or the user is transmitted online.

5. No Tracking

  • No usage analytics
  • No advertising
  • No profiling or telemetry
  • No sharing with third parties

6. Data Security

Data is stored exclusively on the user’s device and/or on the NFC tag, protected by standard operating system security measures.

The user may delete all data at any time by uninstalling the app or resetting/overwriting the NFC tags.

CSV note: any exported CSV file contains only tag names and numeric values chosen by the user; it is saved locally and never uploaded to external services.

7. Use by Minors

The app can also be used by minors as no personal data is collected. In educational or professional contexts, supervision is recommended.

8. Changes to this Policy

Any changes will be made available through an app update and/or this page. The “Last update” date indicates the current version.

9. Contact

For questions or requests you can contact the developer:

Email: fulvio.gilardoni@ammira.it

Privacy Policy TagTally (Italiano)

Ultimo aggiornamento: Novembre 2025

1. Informazioni generali

TagTally è un’applicazione che utilizza la tecnologia NFC per leggere e aggiornare un contatore memorizzato direttamente su tag NFC compatibili (NDEF). L’app è progettata per funzionare offline, senza account e senza servizi cloud.

2. Tipologia di dati trattati

Nessun dato personale viene raccolto, archiviato o trasmesso.

L’app gestisce esclusivamente:

  • Dati memorizzati sul tag NFC: ad esempio un numero intero (contatore) e, facoltativamente, un nome del tag scelto dall’utente.
  • Dati locali sul dispositivo: storico locale essenziale degli utilizzi e preferenze dell’app (es. tema o modalità solo-lettura).

Questi dati rimangono solo sul dispositivo e/o sul tag NFC e non vengono inviati a server esterni o terze parti.

3. Permessi richiesti dall’app

PermessoMotivoTrasmissione dati
NFCLettura e scrittura del contatore sul tagNessuna trasmissione
Vibrazione (Haptic feedback)Feedback tattile durante l’usoNessuna trasmissione
(Facoltativo) Accesso ai file localiSalvataggio/esportazione manuale di file CSVNessuna trasmissione

TagTally non accede a fotocamera, microfono, contatti o localizzazione.

4. Connessione Internet

TagTally non utilizza Internet per le funzionalità operative dell’app.

La connessione viene usata solo per gestire gli acquisti in-app e la verifica licenze tramite Google Play.
Nessun dato relativo ai tag NFC, all’uso dell’app o all’utente viene trasmesso online.

5. Nessun tracciamento

  • Nessun sistema di analisi dell’uso (analytics)
  • Nessuna pubblicità
  • Nessuna profilazione o telemetria
  • Nessuna condivisione con terze parti

6. Sicurezza dei dati

I dati sono conservati esclusivamente sul dispositivo dell’utente e/o sul tag NFC, protetti dalle misure di sicurezza standard del sistema operativo.

L’utente può eliminare tutti i dati in qualsiasi momento disinstallando l’app o resettando/sovrascrivendo i tag NFC.

Nota CSV: l’eventuale file CSV esportato contiene solo nomi dei tag e valori numerici scelti dall’utente; viene salvato unicamente in locale e non è inviato automaticamente a servizi esterni.

7. Utilizzo da parte di minori

L’app può essere utilizzata anche da minori poiché non raccoglie dati personali. In contesti educativi o professionali è consigliata la supervisione.

8. Modifiche a questa Policy

Eventuali modifiche saranno rese disponibili tramite aggiornamento dell’app e/o di questa pagina. La data di “Ultimo aggiornamento” indica la versione corrente.
